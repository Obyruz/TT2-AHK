# TT2-AHK
Automated Script for Tap Titans 2 using AutoHotkey(AHK).

Recommended:
-Nox emulator
-480x800 resolution

Currently Available:
-Simple GUI(start, stop)
-Tap spam(via spacebar, set it up in your preferred Android emulator)
-Leveling up Sword Master
-Prestige after 20 minutes
